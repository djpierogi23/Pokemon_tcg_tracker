# Pokémon TCG Collection Tracker

A modern, interactive web app for tracking your Pokémon Trading Card Game collection.

🌐 **Live at**: [https://djpierogi23.github.io/Pokemon_tcg_tracker/](https://djpierogi23.github.io/Pokemon_tcg_tracker/)

## Features

- 🏠 **Dashboard** — Overview of your entire collection with progress tracking by generation
- 📋 **Set Detail Pages** — View and manage individual cards with one-click HAVE/NEED toggling
- 🔍 **Global Search** — Search across all 20,000+ cards instantly
- 📊 **Statistics** — Completion charts, top sets, and collection analytics
- 💾 **Export/Import** — Backup your data as JSON, import from backups
- 📱 **Responsive** — Works on desktop, tablet, and mobile
- 🌙 **Dark Theme** — Modern dark UI with Pokémon type color coding

## Tech Stack

- Pure HTML / CSS / JavaScript (no frameworks)
- localStorage for data persistence
- Hosted on GitHub Pages

## Data

Your collection data is stored in your browser's localStorage. Changes are auto-saved.
Use the 💾 Import/Export page to backup or restore your data.
